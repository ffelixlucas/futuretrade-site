# Future Trade - Site Institucional

Este é o site oficial da **Future Trade**, desenvolvido por [Lucas Fanha Felix](https://linkedin.com/in/lucasfanhafelix). O projeto foi construído com foco em performance, responsividade e experiência do usuário, utilizando **HTML5**, **CSS3** e **JavaScript** puro.

## 🌐 Acesse o Site

🔗 [https://futuretrade.com.br](https://futuretrade.com.br)

## ✨ Destaques do Projeto

- **Design Responsivo:** Compatível com dispositivos móveis e desktops.
- **Layout Moderno:** Visual limpo e profissional.
- **HTML Semântico e SEO:** Estrutura otimizada para motores de busca.
- **Efeitos Visuais:** Animações suaves e rolagem fluida entre seções.
- **Formulário de Contato:** Validação com feedback visual via SweetAlert2.
- **Menu Hambúrguer Mobile:** Interação fluida com foco em UX.

## 🛠 Tecnologias Utilizadas

- **HTML5**
- **CSS3 (Grid, Flexbox, variáveis personalizadas)**
- **JavaScript**
- **ScrollReveal** – Animações na rolagem
- **Typed.js** – Efeito de digitação dinâmico
- **SweetAlert2** – Feedback visual no formulário
